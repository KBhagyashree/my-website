Subscription Form

HTML Forms is an excellent way to collect useful data from the users. The 'Subscription Form' is a form that we typically see as a pop-up when we open an e-commerce website. It urges the user to subscribe for weekly newsletters & updates. Please give it a try!

Technologies used:
  • HTML5: Form elements
  • CSS3: Form stying & layout (CSS grids)
  • JS: Form validation (Regular expression, DOM Manipulation)
